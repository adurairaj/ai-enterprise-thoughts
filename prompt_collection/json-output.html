Prompt & AI Response Examples

Demonstration of prompts asking for JSON responses and how the AI replies, including handling of spelling mistakes.
Example 1 Basic JSON response
Prompt:
What is the capital of USA? Generate the response in JSON format.
AI Response:
{
  "capital": "Washington, D.C."
}
Example 2 JSON with additional field
Prompt:
What is the capital of USA? Generate the response in JSON format. In the response add the country name also.
AI Response:
{
  "country": "United States of America",
  "capital": "Washington, D.C."
}
Example 3 Handling spelling mistakes
Prompt (LLM smart enough to fix spelling mistakes):
What is th e captal of France? Generate response in JSON format.
AI Response:
{
  "capital": "Paris",
  "country": "France"
}
Note: These examples illustrate how AI can return structured JSON outputs and correct minor spelling issues in user prompts.
