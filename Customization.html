Customize software products using AI. Customization effort are usually very expensive during implementation of new requirements and during upgrade of the products. During upgradation of the product, we need to make sure the old customization are still meaningful and need to be migrated. I want the customization to be done using AI prompts, thus reducing implementation effort. For this to work AI should be aware of the repo, business process, business validations and database entities. Types of customization:
  • Add a column to an entity
  • Delete a column from an entity
  • Add a new entity
  • Delete an existing entity
  • Change business process
  • Add a new validation
  • Modify a validation
